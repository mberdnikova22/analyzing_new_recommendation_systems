## Анализ новых систем рекомендаций с помощью A/B-теста

Компания разрабатывает мобильное приложение, которое включает в себя ленту новостей и мессенджер. Команда машинного обучения разработала новую систему рекомендаций, которая должна показывать пользователям наиболее интересные для них посты, тем самым повышая уровень вовлеченности.
Новый алгоритм рекомендует пользователям посты, которые наиболее похожи на те, которые они лайкали ранее.

Для оценки эффективности новой системы был проведен A/B-тест:
•	Группа 1 была выбрана в качестве контрольной, где использовалась старая система рекомендаций.
•	Группа 2 использовала новый алгоритм рекомендации постов.
Анализируемая метрика: CTR из просмотров в лайки.

Гипотезы:
H0: CTR1 = CTR2 (CTR в контрольной группе равен CTR во второй группе).
H1: CTR1 < CTR2 (CTR в контрольной группе меньше, чем CTR во второй группе).
